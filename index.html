Festi-Friend

The Ultimate Mobile Festival Companion

Festi-Friend is a native Android app developed in Android Studio using the Kotlin programming language. It acts as an essential festival guide, simplifying event planning and attendance for users. The app is supported at the back-end by Firebase.

Key features include:

  • Google Sign-in.
  • Realtime updates.
  • Three-day forecast for the festival location.
  • Comprehensive event schedule that allows the user to curate their own personal timetable.
  • A brief bio of each artist performing at the festival.
  • Community section where users can upload images, review the festival, and engage with posts.
